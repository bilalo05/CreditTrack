# CreditTrack

A complete credit and client management platform consisting of a mobile application and a web administration panel.

## Features

- Client Management
- Credit Tracking
- Payment Tracking
- Outstanding Balance Monitoring
- Statistics Dashboard
- Administrative Web Panel
- Secure Authentication

## Technology Stack

### Mobile
Android (Java)

### Web
Laravel
Vue.js
### Database
MySQL
